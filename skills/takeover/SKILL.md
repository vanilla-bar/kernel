# Takeover - 引き継ぎの読み込みと作業再開

## 注意事項

- コンテキスト汚染を防ぐため、読み込む対象が確定するまでファイル本文を Read しないこと
- 引き継ぎ書が存在しない場合は、その旨をユーザーに伝えること

## 手順

### ステップ1: ファイル特定（本文は読まない）

1. `.claude/handovers/` 内のファイル名一覧を Glob で取得する（なければユーザーに伝えて終了）
2. $ARGUMENTS が指定されていれば一致するファイルを特定する
3. 未指定なら AskUserQuestion で選択UIを表示する（直近4件を options に、最新に「（最新）」付与）

### ステップ2: 読み込みと報告

4. 選択された引き継ぎ書を Read で読み込む（ファイルインデックスに記載されたファイル本体は読まない。タスク着手時に読めばよい）
5. 「引き継ぎ読み込み完了」と伝える。未完了の前提がある場合のみ警告する。

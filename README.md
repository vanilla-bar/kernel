# kernel

Claude Code 向けのプラグインマーケットプレイス。

## インストール

```bash
# 1. マーケットプレイスを追加
/plugin marketplace add vanilla-bar/kernel

# 2. プラグインをインストール
/plugin install ae@kernel
```

## プラグイン一覧

| プラグイン | 説明 |
|-----------|------|
| [ae](plugins/ae/) | Git/GitHub ワークフロー & 開発支援スキル集（15スキル） |
